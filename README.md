# My-calendar

이 프로젝트는 **개인적으로 만든 일정 생성 및 관리 앱**입니다.
- **Electron**을 이용하여 데스크톱 앱으로도 배포할 예정입니다.
- **GraphQL**을 이용해 백엔드와 통신합니다.
- 프론트엔드에서 GraphQL 요청을 처리할 때는 **직접 제작한 라이브러리(`graphql-mini`)**를 사용합니다. (Apollo 등 외부 라이브러리 미사용)
